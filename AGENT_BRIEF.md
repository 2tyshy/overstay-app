## Что строим

**RunLog** — Telegram Mini App для цифровых номадов.

Две вещи которых нет ни у одного конкурента одновременно:
1. Трекер визовых дедлайнов с уведомлениями
2. Community-база реальных схем визаранов — живые данные от таких же людей, не официальные правила

Аудитория: русскоязычные номады в ЮВА с RU/UA/KZ паспортами.
Монетизация: подписка $6-9/мес позже. MVP — бесплатно, собираем аудиторию.

---

## Три экрана

### 1. Статус (главный)
- Большое число дней до дедлайна
- Цветная risk-полоска: зелёная → жёлтая → красная
- Сегментированный прогресс-бар (не линейный, а блоки)
- Статистика: стран / дней в пути / ранов
- Даты въезда и дедлайна
- Алерт если < 14 дней
- 3 кнопки: Схемы / Добавить штамп / PDF-экспорт
- История предыдущих виз

### 2. Схемы
- Фильтры по стране отправления
- Карточки схем: маршрут → КПП → описание → лайфхак → стоимость / время / дата → голосование
- Кнопка "Добавить схему" (форма)

### 3. Дальше
- Баннер с текущим статусом
- Список стран под паспорт пользователя: флаг / название / тип визы / стоимость жизни / количество схем в базе

## Критически важно для UI

**Смотри DESIGN_SYSTEM.md** — там полный design language.

Кратко:
- Nothing OS эстетика: OLED чёрный `#000000`, монохром, один красный акцент
- Шрифты: Geist Sans (текст) + Geist Mono (данные/метки)
- Иконки: Lucide, `stroke-width: 1.5`, `stroke-linecap: round`
- Никаких скруглений > 5px, никаких градиентов, никаких теней
- `-webkit-font-smoothing: antialiased` — обязательно для iOS
- Все числа в Geist с `font-weight: 700`, `letter-spacing: -0.03em`

Референс UI уже сделан — файл `UI_REFERENCE.html`. Построй точно так.

## Чего НЕ делать

- Не добавлять анимации которых нет в референсе
- Не менять цвета (только `#000000` + оттенки белого + `#ff2626`)
- Не использовать другие шрифты
- Не хранить фото паспортов на сервере (только base64 → Claude → выбросить)
- Не добавлять онбординг-экраны пока не попросят
- Не усложнять — MVP должен работать за 10 дней
